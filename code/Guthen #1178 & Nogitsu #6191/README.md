# GNGames

Petite librairie conçue pour créer des mini-jeux sur le Mod de Garry. Conçu pour l'événement [Battle Creators Battle Game Creators Area] (https://g-ca.fr/).

## Commandes client

* `gngames_matchmaking`: Ouvre l'interface de matchmaking

## Commandes serveur

* `gngames_get_all_parties`: Affiche les informations de toutes les parties
* `gngames_reset_parties`: Réinitialise toutes les parties
* `gngames_force_create_party`: Force la création de parties sur les identifiants des joueurs
