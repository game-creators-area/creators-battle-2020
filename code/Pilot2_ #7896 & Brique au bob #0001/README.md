# Color Run

Color Run est un mini jeu de plateforme qui rassemble 4 modes de jeux:

## Modes de jeu

* **Color Shuffle**: Courir et s’arrêter  sur la couleur indiquée en bas de l’HUD
* **Color Conquest**: Obtenir la partie de la plateforme avec le plus de sa couleur à soit
* **Color Tron**: Eviter les couleurs adverses sinon tu meurs
* **Color Fade**: courir pour éviter de tomber dans le trou qui se forme derrière soit
